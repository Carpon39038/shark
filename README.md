# Shark

为速度而生的开源素材管理器。

Shark 是一款本地优先的素材管理工具，可替代 [Eagle](https://eagle.cool/)。面向设计师、插画师以及所有需要管理大量图片的人。没有云端、没有账号、没有订阅——只有你的文件，按你的方式组织。

**[English](docs/README.en.md)**

## 功能特性

- **极速网格浏览** — 虚拟滚动，10 万+素材依然流畅
- **即时搜索** — 基于文件名、标签、备注的全文检索（< 200ms）
- **两级缩略图** — 256px 用于浏览，1024px 按需生成，告别卡顿
- **智能文件夹** — 将筛选条件保存为实时文件夹
- **标签与评分** — 通过标签、评分和备注整理素材
- **去重检测** — 基于 SHA256 内容哈希的导入去重
- **颜色提取** — K-means 聚集算法自动提取主色调
- **本地与隐私** — 一切数据都在你的机器上。SQLite + 文件系统，不会上传任何内容

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18, TypeScript, Zustand, Tailwind CSS |
| 后端 | Tauri v2 (Rust) |
| 数据库 | SQLite（WAL 模式）+ FTS5 |
| 虚拟滚动 | @tanstack/react-virtual |
| 图像处理 | Rust `image` crate |

## 支持格式（MVP）

JPG、PNG、GIF、WebP、SVG、BMP

*后续计划：PSD、AI、Sketch、TIFF、视频、字体*

## 路线图

- **阶段 1** — 核心浏览：素材库管理、文件夹导入、缩略图生成、虚拟网格、图片查看器
- **阶段 2** — 整理组织：标签、文件夹树、拖拽导入、搜索、颜色提取、评分、批量操作
- **阶段 3** — 打磨发布：智能文件夹、去重对话框、错误处理、性能优化、打包与自动更新
